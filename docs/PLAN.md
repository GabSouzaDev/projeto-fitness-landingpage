# Projeto de Landing Page para app Fitness

Página com fundo branco e fonte Roboto como principal.
Objetivo é ter uma landing page clean, branca e sofisticada.

### Seção Header:

Logo no canto esquerdo
- A logo é um F em um círculo (feito com CSS).
- O fundo do círculo é VERDE ESCURO.

Botão de download no canto esquerdo
- Fundo verde, cor branca.
- borda arredondada.

### Seção HERO :
A Hero deve ter altura 90vh, dividio em 3 áreas horizontais.

Esquerda: 
- Apenas a headline, sem ser grande demais, sem destaque (o destaque é o print do app no meio).
"O app perfeito para acompanhar sua vida fitness."

Centro:
- Em destaque a imagem "images/print_1.png que é um print do app.

Direita:
- Uma lista de bullet points sobre o app:
- ❤️ Saúde
- 🏃🏻‍♀️‍➡️ Exercício
- 🤝 Amigos

### Seção de FUNCIONALIDADES:

Vão ser usadas as imagens images/print_2.png, images/print_3.png e images/print_4.png e serão exibidas da seguinte forma:

- print / texto
- texto / print
- print / texto

Ou seja, irá alternar entre o print na esquerda e um texto explicativo na direita e vice-versa.

Descrição breve dos prints:

- print_2.png = Contagem de calorias.
- print_3.png = Exercícios Específicos (corrida, musculação etc.).
- print_4.png = Compartilhamento com amigos.

### Seção de CTA (Call to Action):

Botão grande de download com um texto abaixo dele destacando:

"100% GRATUITO".

### Seção de RODAPÉ:

Apenas com o texto "Todos os direitos reservados - Apple inc."
